LIGHT RAY TYPE 1: VOLUMETRIC SUNLIGHT (God Rays)
================================================================================

CATEGORY: Natural sunlight breaking through atmosphere

PRIMARY USE CASES:
- Outdoor daytime scenes
- Atmospheric mood setting
- Natural environmental lighting
- Depth perception establishment
- Scene ambiance enhancement

VISUAL SPECIFICATIONS:

Ray count: 8-12 rays per scene
- Lower count (8): Subtle, refined look
- Medium count (10): Standard, balanced appearance
- Higher count (12): Rich, detailed volumetric effect

Ray geometry:
- Ray width at origin: 40-80 pixels
- Ray width at end: 200-400 pixels
- Ray length: 400-800 pixels
- Ray shape: Cone/fan spreading outward
- Falloff type: Linear width increase

Ray positioning:
- Origin point: Light source location (sun position)
- Spread angle: 15-30 degrees total fan spread
- Individual ray spacing: Uniform angular distribution
- Angle increment: Total_spread / ray_count

Color specifications:
- Primary color: #FFFFFF (white)
- Secondary color: #FFE58C (light yellow)
- Color blend: 80% white + 20% yellow
- Alpha channel: 0.20-0.35 opacity
- Color variation: ±5% per ray (natural variation)

Animation:
- Movement type: Gentle sway oscillation
- Sway frequency: 0.5-1.0 Hz (slow undulation)
- Sway amplitude: ±8-12 pixels perpendicular to ray
- Pattern: Sine wave motion
- Persistence: Continuous throughout scene

Blend mode:
- Primary blend: Screen (additive lighting)
- Opacity in blend: 0.25-0.35x
- Effect: Bright atmospheric light rays
- Interaction: Additive with existing scene lighting

SHADER SPECIFICATIONS:

Volumetric calculation:
1. Sample along ray from origin to end
2. Sample lighting/shadow information at each sample
3. Accumulate light contribution based on visibility
4. Apply aerial perspective (distant rays dimmer)
5. Apply falloff curve (center bright, edges dim)

Performance optimization:
- Sampling resolution: 32-64 samples per ray (balance quality/performance)
- Downsampling: Optional 50% resolution rendering
- Early termination: Stop sampling if opacity reaches 1.0
- GPU compute: Use compute shaders for efficiency

TEMPORAL STABILITY:
- Frame-to-frame consistency: Critical for smooth appearance
- Noise generation: Per-frame consistent (seeded random)
- Dither pattern: Optional temporal dithering
- Stability target: No visible flickering

INTEGRATION:
- Render pass: Post-processing or deferred lighting
- Z-test: Optional soft Z-testing (depth fade-out)
- Early return: Skip rendering if light source off-screen
- Optimization: Frustum culling by ray extent

SCENE APPLICATION:

Typical usage:
- Applied to: Outdoor scenes with sunlight
- Time of day: Morning or afternoon (sunlight angle moderate)
- Intensity: Scene-dependent (adjust opacity per scene)
- Effect type: Always-on for outdoor scenes

Light source positioning:
- Sun position: Typically off-screen or at screen edge
- Ray origin: Calculated from light direction
- Ray direction: Matches light direction vector
- Adaptation: Adjust for sun position per time of day

USE CASE EXAMPLE:
Scene: Opening establishing shot (Scene 1)
Ray count: 10 rays
Ray length: 600 pixels
Opacity: 0.28x
Sway: 0.6 Hz, ±10 pixels amplitude
Effect: Peaceful morning atmosphere, depth establishment

LIGHT RAY TYPE 2: MAGICAL MYSTICAL RAYS
================================================================================

CATEGORY: Fantasy/supernatural light effects

PRIMARY USE CASES:
- Magical moments
- Creature appearances
- Supernatural events
- Fantasy scene transitions
- Magical power displays

VISUAL SPECIFICATIONS:

Ray count: 5-8 rays per effect
- Lower count (5): Focused, minimal distraction
- Standard count (6): Balanced supernatural feel
- Higher count (8): Rich, complex mystical effect

Ray geometry:
- Ray width at origin: 30-60 pixels
- Ray width at end: 250-500 pixels (wider spread than Type 1)
- Ray length: 600-1200 pixels (longer reach)
- Ray shape: Expanding cone with faster spread
- Falloff: Ease-out curve (nonlinear)

Ray positioning:
- Origin point: Character or magical effect center
- Spread angle: 20-40 degrees (wider than Type 1)
- Pattern: Radial or directional fan
- Rotation: Optional rotating ray pattern

Color specifications:
- Primary color: #FF6B9D (pink)
- Secondary color: #9D4EDD (purple)
- Color blend: 50% pink + 50% purple
- Alpha channel: 0.25-0.40 opacity
- Glow effect: Yes, bloom/glow enhancement applied

Gradient overlay:
- Gradient type: Radial from origin to tip
- Start color: Higher saturation (stronger pink/purple)
- End color: Lower saturation, more transparent
- Gradient falloff: Gaussian curve (smooth)

Animation:
- Movement type: Pulsing + lateral sway
- Pulsing frequency: 2-3 Hz (rapid)
- Pulsing amplitude: ±0.1x opacity (subtle variation)
- Lateral sway: 1-1.5 Hz, ±15-20 pixels
- Pattern: Complex combination of pulse + sway
- Entrance animation: 1.5-3.0 seconds fade-in
- Duration: Persistent or timed (scene-dependent)

Blend mode:
- Primary blend: Screen at 0.6x opacity
- Secondary blend: Additive at 0.2x opacity
- Combined effect: Bright magical glow
- Special: Bloom/glow shader enhancement

PARTICLE ENHANCEMENT:

Floating sparkles along rays:
- Sparkle count: 10-20 per ray
- Sparkle type: Small bright point particles
- Sparkle color: #FFFFFF (white) or color-matching ray
- Sparkle size: 1-3 pixels
- Sparkle movement: Drifting along ray path
- Sparkle lifetime: 0.5-2.0 seconds
- Sparkle animation: Twinkle (opacity pulsing)

SHADER SPECIFICATIONS:

Complex volumetric calculation:
1. Raycasting from origin along ray direction
2. Sample atmospheric density at each point
3. Sample magical field/texture pattern
4. Apply pulsing animation (time-based variation)
5. Apply swaying lateral displacement
6. Accumulate light with bloom contribution
7. Generate sparkle positions along path

Performance optimization:
- Sampling: 48-96 samples per ray (higher quality than Type 1)
- Compute: Use compute shaders for complex calculation
- LOD system: Reduce sample count on lower-end devices
- Caching: Cache ray geometry, recalculate animation only

TEMPORAL STABILITY:
- Frame consistency: High-frequency changes acceptable (magical effect)
- Noise generation: Per-frame varied (intentional animation)
- Stability: No hard transitions between frames
- Effect: Shimmering, dynamic appearance

INTEGRATION:
- Render pass: Deferred lighting + post-processing
- Z-test: Soft Z-testing (no hard depth cutoff)
- Bloom pass: Enhanced bloom for glow effect
- Special effects: Optional additional glow layers

USE CASE EXAMPLE:
Scene: Well encounter mystical moment (Scene 3)
Ray count: 6 rays
Ray length: 800 pixels
Opacity: 0.32x base, 0.35x peak pulse
Sway: 1.2 Hz, ±18 pixels
Pulse: 2.5 Hz, ±0.12x opacity variation
Sparkles: 15 per ray, twinkling
Effect: Magical, supernatural atmosphere

LIGHT RAY TYPE 3: FILTERED INTERIOR LIGHT
================================================================================

CATEGORY: Warm, filtered light in enclosed spaces

PRIMARY USE CASES:
- Indoor scenes
- Cave/enclosed environments
- Sunlight through windows (interior)
- Warm atmospheric lighting
- Intimate scene enhancement

VISUAL SPECIFICATIONS:

Ray count: 6-10 rays per scene
- Lower count (6): Subtle, minimal distraction
- Standard count (8): Balanced interior feel
- Higher count (10): Rich, complex light pattern

Ray geometry:
- Ray width at origin: 50-100 pixels (wider origin)
- Ray width at end: 150-300 pixels (moderate spread)
- Ray length: 300-500 pixels (shorter, contained)
- Ray shape: Slightly spreading cone
- Falloff: Slow linear falloff (maintains brightness)

Ray positioning:
- Origin point: Window or light source interior position
- Spread angle: 10-20 degrees (narrow, focused)
- Pattern: Tight fan pattern
- Direction: Inward into room (or scene-specific)

Color specifications:
- Primary color: #FFA500 (orange)
- Secondary color: #FFD700 (gold)
- Color blend: 60% orange + 40% gold gradient
- Alpha channel: 0.15-0.25 opacity
- Warm tone: Distinctly warm/golden feeling

Color gradient:
- Start (origin): Brighter gold (#FFE580)
- End (far): Warmer orange (#FFA500)
- Transition: Linear color shift along ray
- Effect: Natural sunlight filtering through atmosphere

Animation:
- Movement type: Very slow sway + subtle drift
- Sway frequency: 0.2-0.4 Hz (very slow)
- Sway amplitude: ±4-6 pixels (subtle motion)
- Drift: Slow overall drift (0.5-1.0 px/second)
- Pattern: Nearly stationary, almost imperceptible motion
- Characteristic: Peaceful, undisturbed light

Blend mode:
- Primary blend: Screen at 0.4x opacity
- Effect: Warm atmospheric glow
- Interaction: Blends naturally with scene lighting
- No secondary effects

DUST PARTICLE INTERACTION:
- Dust visibility: Enhanced within ray areas
- Dust color: Slightly warmer tone within rays
- Dust density: May appear slightly increased (optical effect)
- Effect: Dust particles become more visible in warm light

SHADER SPECIFICATIONS:

Simplified volumetric calculation:
1. Raycasting from origin to ray end
2. Sample atmospheric density (uniform for interior)
3. Apply color gradient based on distance
4. Apply slow sway animation
5. Accumulate with minimal complexity

Performance optimization:
- Sampling: 24-32 samples per ray (simple calculation)
- Compute: Can use simple pixel/fragment shaders
- Efficiency: Low performance cost (suitable for all platforms)
- Caching: Ray geometry stable, animate only

TEMPORAL STABILITY:
- Frame consistency: Very high (nearly static)
- Motion: Imperceptible (intentional calm effect)
- Stability: Very smooth, no artifacts
- Effect: Peaceful, undisturbed atmosphere

INTEGRATION:
- Render pass: Can integrate into forward rendering
- Z-test: Optional soft Z-testing
- Bloom: Minimal bloom (warm tone only)
- Special effects: None needed (simple effect)

SCENE APPLICATION:

Well scene considerations:
- Location: Interior well environment
- Light source: Opening/rim of well
- Ray direction: Downward into well
- Lighting setup: Multiple rays from different angles

USE CASE EXAMPLE:
Scene: Well encounter - interior scene (Scene 3, 1:00-1:20)
Ray count: 8 rays
Ray length: 400 pixels
Opacity: 0.20x (dim interior light)
Sway: 0.3 Hz, ±5 pixels
Color: Orange/gold blend
Effect: Warm, enclosed environment atmosphere

LIGHT RAY TYPE 4: DRAMATIC SPOTLIGHT
================================================================================

CATEGORY: Intense focused light for dramatic moments

PRIMARY USE CASES:
- Highlighting important narrative moments
- Dramatic reveals
- Character focus moments
- Story emphasis
- Attention direction

VISUAL SPECIFICATIONS:

Ray count: 3-5 rays (highly focused)
- Lower count (3): Minimal, intense focus
- Standard count (4): Balanced dramatic effect
- Higher count (5): Complex light pattern

Ray geometry:
- Ray width at origin: 60-120 pixels (wide origin)
- Ray width at end: 800-1200 pixels (extreme spread)
- Ray length: 800-1500 pixels (very long reach)
- Ray shape: Rapidly expanding cone
- Falloff: Rapid initial spread, then linear

Ray positioning:
- Origin point: Spotlight/dramatic light source
- Spread angle: 5-15 degrees (very tight, focused)
- Pattern: Tight cone expanding to large area
- Direction: Directional toward subject

Color specifications:
- Primary color: #FFFFFF (pure white)
- Optional color tint: Scene-dependent
- Opacity: 0.35-0.50 (prominent effect)
- Alpha variation: Scene-specific adjustment

Color tint options (scene-dependent):
- Neutral moment: No tint, pure white
- Magical moment: Gold (#FFD700) tint
- Danger moment: Red (#FF0000) tint
- Mystery moment: Blue (#0066FF) tint

Animation:
- Movement type: Optional pulsing intensity
- Pulsing frequency: 2-3 Hz (visible pulsing)
- Pulsing amplitude: ±0.1x opacity
- Duration: 3-10 seconds per spotlight effect
- Entrance: Optional fade-in over 300-500ms
- Exit: Optional fade-out over 500-1000ms

Blend mode:
- Primary blend: Screen at 0.8x opacity (strong effect)
- Effect: Intense bright spotlight
- Interaction: Dominates scene lighting
- Secondary effects: Optional lens flare

LENS FLARE (Optional Enhancement):
- Flare type: Camera lens artifact
- Flare count: 3-5 flare elements
- Flare color: Matching light ray color
- Flare intensity: 30-50% of ray intensity
- Flare positions: Along ray direction (center line)
- Animation: Subtle pulsing (sync with ray)
- Effect: Adds realism/drama

SHADER SPECIFICATIONS:

Complex focused volumetric calculation:
1. Cone-shaped raycasting from spotlight origin
2. Tight angle creates focused beam
3. Rapid spread function creates cone shape
4. Sample atmospheric density at each point
5. Apply pulsing animation if enabled
6. Generate flare artifacts along primary ray
7. Accumulate to final bright spotlight

Performance optimization:
- Sampling: 32-64 samples per ray (moderate quality)
- Compute: Use compute shaders for complex cone calculation
- LOD: Reduce sample count on performance-constrained devices
- Early exit: Can skip rendering outside spotlight area

TEMPORAL STABILITY:
- Frame consistency: High (dramatic effect should be stable)
- Motion: Pulsing animation (intentional, smooth)
- Stability: Very smooth transitions
- Flare: Subtle, doesn't distract from effect

INTEGRATION:
- Render pass: Deferred or post-processing
- Z-test: Optional soft Z-testing
- Bloom: Enhanced bloom for bright effect
- Priority: High-priority effect (renders last)

USE CASE EXAMPLE:
Scene: Climactic finale key moment (Scene 5, 2:55-3:00)
Ray count: 4 rays
Ray length: 1000 pixels
Opacity: 0.42x
Pulsing: 2.5 Hz, ±0.08x variation
Tint: Gold (#FFD700)
Effect: Dramatic focus on transformation moment

HAZE TYPE 1: LIGHT FOG (Ambient Background)
================================================================

CATEGORY: Subtle atmospheric depth enhancement

SPECIFICATIONS:
- Coverage: 10-20% of screen
- Density: Very light, transparent
- Color: #E8E8E8 to #B0C4DE gradient (light gray to steel blue)
- Opacity: 0.08-0.15
- Layer depth: Background (behind characters)

VISUAL CHARACTERISTICS:
- Appearance: Very subtle, almost imperceptible
- Motion: Very slow drift (0.5 px/frame horizontal)
- Density variation: Random 5-10% per frame
- Texture: Soft noise pattern (Perlin noise preferred)
- Depth feeling: Adds atmospheric perspective

SHADER SPECIFICATIONS:
- Method: Procedural noise-based fog
- Noise type: Perlin noise (2D or 3D)
- Sample resolution: Low (32x32 texture typical)
- Animation: Drift texture coordinates slowly
- Blend: Normal mode at low opacity

ANIMATION:
- Horizontal drift: Slow, continuous (0.5 px/frame)
- Vertical drift: Optional subtle (±0.2 px/frame)
- Density pulsing: Very subtle (±2% variation)
- Frequency: 0.3 Hz pulsing (barely noticeable)

USE CASES:
- Ambient atmospheric enhancement
- Outdoor depth establishment
- General background enhancement
- Used in most scenes for subtle effect

BLEND MODE:
- Mode: Normal
- Opacity: 0.08-0.15x
- Color: Light gray (#E8E8E8) primary

SCENE APPLICATION:
- Persistent background effect
- Opacity adjustable per scene (0.08-0.15 range)
- Can be toggled on/off based on performance
