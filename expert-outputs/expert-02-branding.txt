## 2. 📌 YouTube Handle Convention Recommendations (2026 Format)

### Handle Rules in 2026:
- Must be **3–30 characters**, letters/numbers/underscores/hyphens only
- Case-insensitive but **display case matters** for branding
- Handles are now **primary discovery URLs**: `youtube.com/@YourHandle`

### Best Practice Framework:

```
TIER 1 — Ideal Handle (≤15 chars, no separators)
  @EchoedMinds       ✅ Clean, memorable, no ambiguity
  @TheHollowHour     ✅ Slightly long but cadence is perfect

TIER 2 — Acceptable (with separator for readability)
  @Unspoken_Archive  ✅ Underscore separates compound words
  @VeilAndVoice      ✅ "And" reads naturally

TIER 3 — Avoid
  @echoed.minds      ❌ Dots not allowed on YouTube
  @EchoedMinds2024   ❌ Year dates your brand instantly
  @TheRealEchoedMinds ❌ "TheReal" signals you're not first
```

### Handle SEO Tips (2026):
- **Match across all platforms**: Same handle on TikTok, Instagram, X — reserve all simultaneously
- **No underscores if possible** — verbal sharing is cleaner ("at Echoed Minds" vs "at Echoed underscore Minds")
- **Check handle on**: namecheckr.com before committing

## 3. 🎨 Logo Design Guidance

### Design Philosophy for Faceless Channels:
> *Your logo IS your face. It must carry personality, emotional weight, and be readable at 88×88px (YouTube avatar circle crop).*

### Study of Reference Channels:

| Channel | Logo Strategy | What to Steal |
|---|---|---|
| **MagnatesMedia** | Bold serif wordmark + heavy drop shadow; gold/black | Confidence through typography weight |
| **Kurzgesagt** | Flat illustration icon + custom sans-serif wordmark; always unified | Icon + wordmark lockup; consistent illustration style |
| **RSlash** | Simple letter mark ("R/") + clean sans-serif | Clever symbol integration (Reddit "R/" becomes the brand) |

### Your Logo System — 3 Components:

#### A. Primary Mark (Icon — use alone for avatar)
```
CONCEPT: Abstract "speech wave" or "open eye"
  — Speech wave = narration across all content
  — Open eye = "see the unseen" (horror, history, Reddit secrets)

FORM: Single geometric shape, no more than 3 anchor points
  — Works as: app icon, watermark, favicon, channel avatar
  — Must READ at 88×88px AND 16×16px

AVOID: Human faces, realistic illustrations, gradients in icon
```

#### B. Wordmark (Text Logo)
```
FONT DIRECTION:
  Option A — Serif + Weight contrast
    "ECHOED" [bold slab serif] + "MINDS" [light weight same font]
    Creates visual tension = memorable

  Option B — Geometric Sans (Futura-style)
    All caps, tracked wide (+150 letter-spacing)
    Clean, premium, YouTube-native

  Option C — Custom ligature
    Connect two letters (E+M for Echoed Minds)
    Unique, ownable, designer-level with free tools

FONTS (Free/Commercial OK):
  • Playfair Display Bold (serif, atmospheric)
  • Space Grotesk (modern, clean)
  • Cinzel (historical gravitas)
  • Raleway ExtraBold (motivational energy)
```

#### C. Lockup Rules
```
HORIZONTAL LOCKUP: [Icon] [Wordmark]  — for banners, thumbnails
STACKED LOCKUP:    [Icon above]       — for avatar, watermark
                   [Wordmark below]
ICON ONLY:         [Icon]             — for watermark overlay on videos

CLEAR SPACE RULE: Minimum 1× icon height of white space on all sides
MINIMUM SIZE: Never use wordmark below 120px wide
```

### Logo Do's and Don'ts:

| ✅ DO | ❌ DON'T |
|---|---|
| Single color + transparent background | Drop shadows on the icon itself |
| Works in white AND black versions | More than 2 typefaces |
| Geometric, scalable shapes | Clip art or stock icon libraries |
| Design for circle crop (YouTube avatar) | Thin strokes under 2px at small sizes |
| Export as SVG (vector) | Use only PNG (pixelates when scaled) |

### Banner Layout Blueprint:

┌──────────────────────────────────────────────────┐

└──────────────────────────────────────────────────┘

  • Dark gradient or subtle texture
  • Keep edges (dead zones) tasteful but non-critical
  • Consider: dark fog texture, paper grain, subtle geometry

| Device | Banner Display Size |
|---|---|
| Desktop browser | 2560×423px (letterboxed) |
| Tablet | ~1855×423px |
| Mobile | ~1546×423px (safe zone only) |
| TV | Full 2560×1440px |

> ⚡ **Pro tip**: Design in the 1546×423 safe zone FIRST. Then extend the background texture to the full 2560×1440 canvas. Never put text outside the safe zone.

## 6. 🎨 Recommended Color Palette

### Design Challenge:
> Must simultaneously work for **Horror** (dark, tense, unsettling) AND **Motivational** (energizing, warm, uplifting) — without feeling like two different brands.

### Solution: The "Deep Ember" Palette

```
PRIMARY COLORS (Brand Core):
┌─────────────────────────────────────────────────────┐
│  ██████  VOID BLACK     #0D0D0D   — Backgrounds     │
│  ██████  IRON GREY      #2A2A2A   — Cards, panels   │
│  ██████  BONE WHITE     #F0EDE8   — Primary text     │
└─────────────────────────────────────────────────────┘

ACCENT COLORS (Playlist-specific overlays):
┌─────────────────────────────────────────────────────┐
│  ██████  EMBER AMBER    #E8942A   — Motivational     │
│          (warm, fire, energy — thumbnail accents)   │
│                                                     │
│  ██████  CRIMSON DUSK   #9B2335   — Horror           │
│          (blood-adjacent but not cheap — premium)   │
└─────────────────────────────────────────────────────┘
```

### How the Palette Functions Per Playlist:

| Playlist | Background | Text | Accent |
|---|---|---|---|
| Reddit Narration | `#0D0D0D` | `#F0EDE8` | `#E8942A` |
| Horror | `#0D0D0D` | `#F0EDE8` | `#9B2335` |
| Motivational | `#2A2A2A` | `#F0EDE8` | `#E8942A` |
| History | `#2A2A2A` | `#F0EDE8` | Split: amber + crimson |

### Why This Works:
- `#0D0D0D + #F0EDE8` = **premium contrast**, not harsh white-on-black
- `#E8942A` (amber) = universal warmth — fire, candlelight, ancient manuscripts (works for ALL 4 niches)
- `#9B2335` (crimson) = reserved for Horror only = **Pavlovian thumbnail recognition**
- The palette is **dark-native** = perfect for YouTube's dark mode (default for most viewers in 2026)

### Color Usage Rules:
```
LOGO:        Bone White #F0EDE8 on dark  OR  Void Black #0D0D0D on light
THUMBNAILS:  Max 2 colors per thumbnail + 1 accent
BANNER:      Void Black bg + Bone White text + Ember Amber accent
NEVER:       Neon colors, pure white (#FFFFFF), pure black (#000000)
```
